# Ejercicios de Repetición

1. Una pirámide de números. Dado un número entero n, imprimir una pirámide de números de tal manera que el último número impreso en una línea correspona al número de linea.
2. Realizar un algoritmo para dibujar un cuadrado de una longitud dada en un rectángulo de longitud 80 y altura 25. El cuadrado debe quedar centrado con respecto al rectángulo.
3. Factorizar un número, suponiendo que el último primo es 17.
