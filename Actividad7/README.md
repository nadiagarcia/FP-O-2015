# Ejercicios de Repetición

1. (2 puntos) Un ciclo que sume los primeros _n_ números
2. (4 puntos) Un algoritmo que encuentre y muestre los números de 4 cifras que cumpla con que la suma de sus cifras impares sea igual a la suma de sus cifras pares.
3. (2 puntos) Un algoritmo que visualice una pirámide como la del ejemplo para cualquier número entero _n_.
4. (2 puntos) Calcular la suma de un conjunto de números dados; el último número del conjunto es cero.
5. (10 puntos) Determinar si un número dado es primo o no.
6. (12 puntos) Calcular y mostrar todos los números primos en rango 1 - 1000.
